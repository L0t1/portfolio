
# Portfolio v2

An interactive, scroll-driven portfolio for Lot Kyçyku — Fullstack Engineer (Backend-Heavy) — built with React, TypeScript, and Vite. Showcases real-world projects, technical skills, and a modern developer journey.

## Features
- Animated hero and mascot
- Backend-heavy technical arsenal
- Timeline-based experience section
- Project highlights with tech stack
- Responsive, modern UI

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

**Prerequisites:** Node.js (v18+ recommended)

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
